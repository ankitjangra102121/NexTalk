# NexTalk

A modern real-time communication platform built with React, Express.js, Socket.IO, Prisma, and PostgreSQL.

https://github.com/ankitjangra102121/NexTalk

---

## ✨ Features

* Real-time messaging powered by Socket.IO
* Secure JWT Authentication & Authorization
* Private one-to-one conversations
* Persistent chat history with PostgreSQL
* Prisma ORM for database management
* Modern responsive UI with Tailwind CSS
* User search and conversation management
* Production deployment on Render

## 🔧 Local Setup

### Clone Repository

```bash
git clone https://github.com/ankitjangra102121/NexTalk.git
cd NexTalk
```

### Backend Setup

```bash
cd server
npm install
npm run dev
```

### Frontend Setup

```bash
cd client
npm install
npm run dev
```
